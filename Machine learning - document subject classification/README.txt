Machine learning - document subject classification
Summary: Built software that used machine learning and text processing to classify news articles according to there topics. The performance of two machine learning algorithms, K-NN and Naïve Bayes, where compared. Different text preprocessing methods where also used.
Languages and libraries used: Python, Numpy, Sklearn
Key skills and algorithms used: K-NN, Naïve Bayes, text processing
